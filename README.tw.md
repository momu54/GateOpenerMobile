# 閘門開門器手機 App

這是一個可以透過網路及微控制器控制閘門的 App
雖然這是用 React Native 寫的但是他並不支援 iOS

# 執行開發伺服器

1. 安裝依賴

    ```sh
    pnpm i
    ```

2. 預先構建

    ```sh
    pnpm prebuild
    ```

3.  - 請透過 ADB 連結
    - Start
        ```sh
        pnpm android
        ```
